# MVP Palestra Argentina

Fecha objetivo: 28 de mayo de 2026.

## Version 0.1

La primera version se distribuye como APK descargable desde una web simple. No requiere cobrar a usuarios ni pagar cuentas de tiendas.

## Pantallas

- Inicio publico con avisos nacionales.
- Agenda publica y agenda interna.
- Materiales publicos, internos y reservados.
- Comunidades por provincia.
- Perfil, registro e inicio de sesion.
- Aprobacion de usuarios y cambio de rol en una segunda pantalla administrativa.

## Roles iniciales

- Miembro.
- Coordinador local.
- Coordinador provincial.
- Asesor.
- Equipo nacional.
- Administrador.

## Permisos iniciales

- Ver materiales internos.
- Ver documentos confidenciales.
- Crear eventos.
- Aprobar usuarios.
- Enviar notificaciones.
- Gestionar contenido.

## Provincias iniciales

- Salta.
- Jujuy.
- Tucuman.
- Catamarca.
- Cordoba.
- San Luis.

## Estrategia gratuita

- Expo y React Native para construir la app.
- Supabase en plan gratis para usuarios, base de datos y archivos.
- APK por descarga directa al inicio.
- Google Play queda para cuando se pueda pagar la cuenta de desarrollador.

## Pendientes inmediatos

1. Instalar Node.js LTS completo.
2. Crear proyecto gratis en Supabase.
3. Cargar `supabase/schema.sql` en el editor SQL de Supabase.
4. Reemplazar el login demo por autenticacion real.
5. Cargar identidad visual oficial: logo, colores y textos institucionales.
6. Generar APK de prueba.
